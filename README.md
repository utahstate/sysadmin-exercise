# sysadmin-exercise:

## Prerequisites

Before you start, ensure that you have:

- A virtual machine (VM) with your desired operating system installed.  You should be able to use Windows or Linux.
- Use the virtualization software of your choice (i.e. VMware, VirtualBox, QEMU, Parallels, etc.)

## Instructions

Setup a webserver on your operating system of choice (i.e. IIS or Apache on Windows or Linux)

Setup git on your machine.  Pull the code for the website from https://github.com/utahstate/sysadmin-exercise-webapp.  View the website.
   
## Presentation and Interview Day

Be prepared to walk us through the methods you used to build the virtual machine and webserver and any further investigation.
