# sysadmin-exercise:

## Prerequisites

Before you start, ensure that you have a virtual machine (VM) with your desired operating system installed.  You should be able to use Windows or Linux.


## Instructions

Setup a webserver on your operating system of choice (i.e. IIS or Apache on Windows or Linux)

Setup git on your machine.  Clone the git repo for the website from `https://github.com/utahstate/sysadmin-exercise-webapp`.  View the website on your webserver.
   
## Presentation and Interview Day

Be prepared to walk us through the methods you used to build the virtual machine and webserver and any further investigation.
